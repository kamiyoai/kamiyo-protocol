# Kizuna X Posts Calendar (12 Weeks)

## Posting Cadence
1. Two posts per week.
2. Post A = education.
3. Post B = proof.

## Post Structure Template
1. One-line problem.
2. One-line Kizuna value statement.
3. One concrete proof point.
4. One next-step CTA.

## Quality Bar
1. Avoid internal terms.
2. Keep each post understandable by non-developers.
3. Prefer one clear claim over many weak claims.
4. Every proof post must include one measurable signal.
