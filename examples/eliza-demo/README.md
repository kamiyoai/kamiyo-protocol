# Kamiyo ElizaOS Demo

```bash
npm install
cp .env.example .env
npm run dev
```

## Environment

| Variable | Default |
|----------|---------|
| `SOLANA_PRIVATE_KEY` | - |
| `KAMIYO_NETWORK` | `devnet` |
| `KAMIYO_QUALITY_THRESHOLD` | `80` |
| `KAMIYO_MAX_PRICE` | `0.01` |
| `KAMIYO_MIN_REPUTATION` | `60` |
