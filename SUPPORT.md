# Support

## Where to Ask Questions

- Use GitHub Issues for bugs and feature requests.
- For security issues, follow `SECURITY.md` and report privately.

## What to Include in Bug Reports

- Component (`programs/`, `crates/`, `services/`, `packages/`)
- Environment details (OS, versions, toolchain)
- Reproduction steps
- Expected behavior vs actual behavior
- Relevant logs or error output

## Maintainer Expectations

Best-effort support is provided for actively maintained areas of the repository. Response times vary based on issue severity and maintainer availability.
